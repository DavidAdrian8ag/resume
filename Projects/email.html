Email Simulation

This is a form used very often on websites, to fill a form, in this case I decided to create a “contact us” page which will “send” emails but with the feature that on some fields like “email”, it must have a valid email, also, it has a trigger to let the person know which fields are mandatory. Finally, when everything is properly filled, the “send” button will activate and will simulate the email being sent. As normal on these formularies, a reset button is placed if you want to delete everything.
